# Marketing-Campaign
**Overview** 

This project focuses on analyzing marketing campaign performance across channels, regions, and key performance metrics.
The dashboard is designed as a hands-on learning project, helping learners practice how to measure Marketing ROI in a clear and data-driven way, while supporting decision-making for budget optimization and communication strategy.

**Objectives**

Evaluate campaign effectiveness by channel and region

Identify high-performing and underperforming marketing activities

Support data-driven decisions for budget allocation and strategy optimization

Practice real-world marketing analytics and performance measurement

# Design Thinking Approach

This project follows a Design Thinking framework to ensure the analysis is user-centered and actionable:

**1. Empathize**
Understand the needs of marketers and decision-makers who need quick, reliable insights into campaign performance.

**2. Define**
Identify key business questions such as:

Which channels deliver the highest ROI?

Which regions perform below expectations?

Where should marketing budgets be reallocated?

**3. Ideate**
Design metrics, KPIs, and dashboard views that clearly reflect campaign efficiency and impact.

**4. Prototype**
Build an interactive dashboard that visualizes performance by channel, region, and KPI.

**5. Test & Iterate**
Validate insights, refine visualizations, and improve clarity to better support decision-making.
